# Intergalactic-Data-Processing-Center
C library for coordinate conversion and telemetry data analysis.

## Quick Start
```bash
git clone https://github.com/d-IT-tO/Intergalactic-Data-Processing-Center.git
cd Intergalactic-Data-Processing-Center
make all
make test
```

## Features

### Module 1: Coordinate Decoder
- Convert coordinates between number systems (base 2-16)
- Support for integer and fractional parts
- Error checking and validation

### Module 2: Telemetry Analyzer  
- Process arrays of numbers in different bases
- Statistical analysis (mean, median, min/max)
- Bubble sort implementation

## Build
```bash
make all        # Build everything
make test       # Run tests
make clean      # Clean build artifacts
```

## Requirements
- C99 compatible compiler GCC
- Make
- Linux with command line
